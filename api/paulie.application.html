paulie.application.otoc.average_otoc(generators, v, w)¶

Computes the Haar averaged out-of-time-order correlator F between two Pauli strings W and V. We write it as E_U[F(W, U^H @ V @ U)] = E_U[tr[W @ U^H @ V @ U @ W @ U^H @ V @ U]] / d where U sampled according to the Haar measure on the Lie group corresponding to the DLA generated by generators and d is the dimension of the Hilbert space.

We can compute this via the commutator graph as 1 - 2 * |{W, P} = 0 : P in connected component of V| / |connected component of V| (arXiV:2502.16404)

Parameters:

  • generators – Generating set of the Pauli string DLA.

  • v – Pauli string V

  • w – Pauli string W

Returns:

Haar averaged out-of-time-order correlator F between two Pauli strings W and V.
