paulie.application¶

Modules

animation

Animation building transformation anti-commutation graph

average_pauli_weight

Module to compute the average Pauli weight (influence) and quantum Fourier entropy of an operator O.

charges

Non-commuting charges describe non-Abelian symmetries — that is, elements of the stabilizer of the DLA that do not commute with each other.

fourpoint

Compute the average out-of-time-order correlator between two Pauli strings.

get_optimal_su2_n

Get generator set with optimal generation rate for su(2**n)

graph_complexity

Computes the average graph complexity of a specified PauliString.

matrix_decomposition

Fast Pauli basis matrix decomposition algorithm.

otoc

Compute the average out-of-time-order correlator between two Pauli strings.

pauli_compiler

The algorithm is based on "Optimally generating su(2^N) using Pauli strings" [1].

pauli_compiler_optimal

Optimal PauliCompiler

plot

Plot anti-commutation graph after transforming graph to a canonical type

second_moment

Module for computing the second-order twirl of an operator over a group generated by system generators, based on the theory in arXiv:2502.16404.
